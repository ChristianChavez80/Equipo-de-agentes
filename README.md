# 🐝 Abelha Studio - Equipo de Agentes

## Bienvenido al Centro de Operaciones

Este es el repositorio central de **Abelha Studio**, una agencia de tecnología web especializada en **Sostenibilidad Digital + Accesibilidad + Automatización**.

---

## 🎯 ¿Qué Somos?

Abelha Studio demuestra que la eficiencia digital, la sostenibilidad ambiental y la inclusión social **no son metas opuestas**, sino los motores de un negocio rentable.

### Tres Pilares de Diferenciación

1. **🌱 Sostenibilidad** (Low-Carbon Web)
   - Sitios 50% más ligeros
   - Infraestructura de energía renovable
   - Reportes de impacto ambiental

2. **♿ Accesibilidad** (WCAG 2.1 AA)
   - Navegable por personas con discapacidades
   - Semántica HTML5 perfecta
   - Certificados de cumplimiento

3. **🤖 Automatización**
   - Flujos CRM-WhatsApp-Email
   - Agentes de gestión de leads
   - Ahorro de 10+ horas/semana por cliente
