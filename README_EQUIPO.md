## 🎪 Cómo Funcionan Los Agentes

### **Ciclo de Colmena: 5 Fases de Proyecto**

```
DIAGNÓSTICO → ARQUITECTURA → CONSTRUCCIÓN → CERTIFICACIÓN → POLINIZACIÓN
(Semana 1-2)  (Semana 3-6)  (Semana 7-10)  (Semana 11)     (Semana 12+)
```

1. **DIAGNÓSTICO** (Research + Estrategia)
   - Investigador: Analiza nicho del cliente
   - Especialista: Define oportunidades
   - Orquestador: Crea plan de 12 semanas

2. **ARQUITECTURA** (Diseño + Estrategia)
   - UX/UI: Prototipos en Figma
   - Copy: Mensajería de marca
   - Inclusive: Validación de accesibilidad

3. **CONSTRUCCIÓN** (Dev + Eco)
   - Backend: APIs y integraciones
   - Fullstack: Estructura de datos
   - Ecology: Optimización de performance

4. **CERTIFICACIÓN** (QA)
   - QA: Testing exhaustivo
   - Inclusive: Auditoría final de accesibilidad
   - Orquestador: Sign-off para lanzamiento

5. **POLINIZACIÓN** (Marketing + Mantenimiento)
   - SEO: Gestión de Google Business Profile
   - Copywriting: Content marketing
   - Finances: Seguimiento de KPIs
