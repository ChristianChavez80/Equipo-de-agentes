# 📦 Servicios Ofrecidos - Abelha Studio

## Cuatro Pilares de Servicio

---

## 🗺️ 1. SEO Local

### Descripción
Posicionamiento de pequeños negocios locales en Google Maps y búsqueda orgánica.

### Entregables Específicos
- Optimización de Google Business Profile (GBP)
- Estrategia de citas locales (directorios)
- Investigación y targeting de palabras clave geolocalizadas
- Estrategia de generación de reseñas (ética)

### Valor Agregado Abelha
- **Reporte de "Presencia en el Mapa"**: Dashboard visual del posicionamiento en Google Maps
- **Estrategia de Reseñas Éticas**: Sistema para pedir reseñas sin spam
- **Seguimiento mensual**: Métrica clara de mejora (% de visibilidad local)

### Duración Típica
- Setup inicial: 2-3 semanas
- Resultados visibles: 6-8 semanas
- Manutención: 4 horas/mes

### Precio Estimado
- Inicial: $400-600 USD
- Mensual: $150-250 USD

---

## 🌐 2. Web Inclusiva

### Descripción
Desarrollo de landing pages o sitios corporativos con alta accesibilidad y rendimiento.

### Entregables Específicos
- Diseño responsivo mobile-first
- Modo oscuro nativo (CSS `prefers-color-scheme`)
- Optimización de Core Web Vitals (LCP, FID, CLS)
- Certificado de cumplimiento WCAG 2.1 AA
- Auditoría técnica de velocidad

### Valor Agregado Abelha
- **Certificado de Accesibilidad**: Documento legal de conformidad
- **Auditoría Core Web Vitals**: Informe detallado de rendimiento
- **Código limpio**: 0 warnings en auditorías automáticas
- **SEO-Ready**: Semántica HTML5 perfecta para indexación

### Duración Típica
- Proyecto: 4-8 semanas (según complejidad)
- Landing simple: 4 semanas
- Sitio corporativo: 6-8 semanas

### Precio Estimado
- Landing page: $1,500-2,500 USD
- Sitio corporativo (5-10 páginas): $3,000-5,000 USD

---

## 🤖 3. Automatización

### Descripción
Flujos de trabajo automatizados entre herramientas de negocio (CRM, WhatsApp, Email).

### Entregables Específicos
- Integración CRM ↔ WhatsApp (notificaciones automáticas)
- Flujo de leads: Formulario → CRM → Email de bienvenida → WhatsApp de seguimiento
- Automatización de citas/reservas
- Reportes automáticos por email
- Webhook personalizado para casos especiales

### Valor Agregado Abelha
- **Ahorro cuantificable**: "Te devolvemos X horas/semana"
- **Agentes de gestión de leads**: Chatbots que califican leads automáticamente
- **Zero-touch sales funnel**: El lead se nutre sin intervención humana hasta que está listo
- **Documentación de flujos**: Cada automatización es explicada paso a paso

### Duración Típica
- Setup simple: 1-2 semanas
- Sistema completo: 4-6 semanas
- Integración y testing: 1 semana

### Precio Estimado
- Automatización básica: $800-1,200 USD
- Sistema completo con agentes: $2,000-4,000 USD
- Mantenimiento mensual: $200-500 USD

---

## 🌱 4. Mantenimiento Eco

### Descripción
Servicio recurrente de mantenimiento técnico + sostenibilidad digital.

### Entregables Específicos
- Actualizaciones de seguridad (plugins, dependencias, core)
- Limpieza de base de datos (tablas no usadas, índices)
- Backup automático diario
- Monitoreo de uptime 99.9%
- Reporte mensual de sostenibilidad

### Valor Agregado Abelha
- **Métrica de Huella de Carbono**: Kilogramos CO2 ahorrados cada mes
- **Velocidad monitoreada**: Alertas si Core Web Vitals se degrada
- **Proactividad**: Detectamos problemas antes de que el cliente lo note
- **Tranquilidad**: Todo está cubierto, cero sorpresas

### Duración Típica
- Contrato mínimo: 3 meses
- Típico: 12 meses
- Indefinido si el cliente desea

### Precio Estimado
- Sitio pequeño (< 50 páginas): $150-250 USD/mes
- Sitio mediano (50-200 páginas): $250-400 USD/mes
- Sitio grande (> 200 páginas): $400-800 USD/mes

## 🎯 Matriz de Decisión para Cliente

### ¿Quiero ser encontrado? → SEO Local
### ¿Quiero una web moderna? → Web Inclusiva
### ¿Quiero ahorrar tiempo? → Automatización
### ¿Quiero tranquilidad? → Mantenimiento Eco

### Combinaciones Recomendadas

**Pequeño negocio local (Restaurante, Salón)**
- SEO Local + Web Inclusiva = $2,500-3,500 iniciales + $300/mes

**Empresa de servicios (Contable, Consultor)**
- Web Inclusiva + Automatización = $3,500-5,000 iniciales + $300-500/mes

**E-commerce**
- Web Inclusiva + Automatización + Mantenimiento Eco = $4,000-6,000 iniciales + $600-1,000/mes

**Empresa consolidada**
- Todos los servicios = Propuesta a medida
